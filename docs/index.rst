Welcome to OSeMBE's documentation!
==================================

The OSeMBE model aims to provide a simple structured, open source European long-term energy planning model. The goal is to provide a platform for collaborative use and starting point for energy systems analysis.

This documentation is under development. Suggestions for needed additions and updates much apprecitaed. Ideally via an issue on the [GitHub repository of OSeMBE](https://github.com/KTH-dESA/OSeMBE).
